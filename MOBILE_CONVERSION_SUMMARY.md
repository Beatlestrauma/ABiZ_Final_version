## Deployment Options

### Option 1: Debug APK (Development/Testing)
- Generated automatically by build script
- Can be shared for testing
- Not suitable for app store

### Option 2: Release APK (Production)
```bash
# 1. Create signing key (one-time)
keytool -genkey -v -keystore release.keystore -alias release -keyalg RSA -keysize 2048 -validity 10000

# 2. Build release
cd android
./gradlew assembleRelease
cd ..
```

### Option 3: Google Play Store
1. Create developer account ($25)
2. Upload release APK
3. Fill metadata (screenshots, description, etc.)
4. Submit for review
